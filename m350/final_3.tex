\section*{3.}
Suppose $v$ is an eigenvector with eigenvalue $\lambda$. Then
$$TUv = \lambda v$$
$$UTUv = U(\lambda v) = \lambda Uv$$
Since matrix multiplication is both left and right associative,
$$(UT)(Uv) = \lambda (Uv)$$
Thus for $w = Uv$
$$UTw = \lambda w$$
and we are done $\Box$
