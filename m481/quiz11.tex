\section{}
For $X_i$ i.i.d. Normal, $\frac{(n-1)s^2}{\sigma^2} \sim \chi^2_{n-1}$.\\
If $s^2 > \sigma_0^2$, $p = 2Pr(\chi^2_{n-1} > \frac{(n-1)s^2}{\sigma^2})$.\\
If $s^2 < \sigma_0^2$, $p = 2Pr(\chi^2_{n-1} < \frac{(n-1)s^2}{\sigma^2})$ $\Box$
